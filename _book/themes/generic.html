Generic

ITRE Generic Theme

The Institute for Transportation Research and Education (ITRE) is an institutional center located at NC State University. The Research Theme is a Wordpress theme for displaying project abstracts with associated author bios and relevant documents.

Theme Requirements

Plugin Version Website
ACF Pro 5.3.8 ACF
NC State Utility Bar 0.1 NC State University

Features

  • Basic theme to match brand standards with NCSU
  • Font Page template with two column and wide widget areas
  • Staff Page with image and links to resume
  • Flexible Page Builder template using ACF fields
  • WCAG 2 AA Accessibility standards
  • Optional Color Schemes with Wordpress customizer
  • Responsive and mobile ready

Sage

This theme was created using the Sage boilerplate from Roots.

Features

  • gulp build script that compiles both Sass and Less, checks for JavaScript errors, optimizes images, and concatenates and minifies files
  • BrowserSync for keeping multiple browsers and devices synchronized while testing, along with injecting updated CSS and JS into your browser while you're developing
  • Bower for front-end package management
  • asset-builder for the JSON file based asset pipeline
  • Bootstrap
  • Theme wrapper
  • ARIA roles and microformats
  • Posts use the hNews microformat
  • Multilingual ready and over 30 available community translations

Setup

Edit lib/setup.php to enable or disable theme features, setup navigation menus, post thumbnail sizes, post formats, and sidebars.

Documentation

Sage documentation is available at https://roots.io/sage/docs/.

Colors

Color schemes can be selected through the Wordpress Customizer interface. Custom color schemes can be defined, although the colors may appear slightly different due to accessibility formulas in the theme. Overriding these changes may cause the theme to no longer meet accessibility standards.
